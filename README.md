# Python Training Practice Tasks

## Project Overview

This repository contains the Python practice tasks assigned during the Python training program.
The purpose of these tasks is to practice and understand fundamental Python programming concepts such as operators, user input handling, string manipulation, and conditional statements.

These exercises help in developing logical thinking and problem-solving skills using Python programming.

---

## Topics Covered

The tasks included in this repository are categorized based on different Python concepts:

### 1. Bitwise Operator Programs

Programs demonstrating the use of bitwise operators such as:

* AND (&)
* OR (|)
* XOR (^)
* Left Shift (<<)
* Right Shift (>>)

### 2. String Handling Programs

Programs that perform operations on strings such as:

* String concatenation
* String manipulation
* Taking string input from the user

### 3. User Input and Type Conversion

Programs that take user input using the `input()` function and convert the data into required data types such as integers.

### 4. Unit Digit Programs

Programs that extract or work with the unit digit of a number using arithmetic operations.

### 5. Conditional Statements (If Programs)

Programs that perform decision making using **if conditions**.

### 6. If–Else Programs

Programs that use **if–else statements** to execute different blocks of code based on conditions.

### 7. Nested If Programs

Programs that contain **multiple levels of if conditions**.

### 8. Match Case Programs

Programs that demonstrate the use of Python's **match-case statement** to perform actions based on user input.
